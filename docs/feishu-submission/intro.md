# 证证鸽

## 让疑似侵权页面，更快变成可行动的证据

**项目定位**

证证鸽是一套面向知识产权侵权响应场景的智能取证与处置辅助平台。  
它把分散、易失的网页线索，快速转化为**标准化证据包、结构化案件材料、风险摘要和可审核的文书初稿**。

**这次作品最想解决的问题**

- 侵权线索被发现后，往往来不及留证
- 证据分散在截图、链接、聊天记录里，难以进入后续法务流程
- 监测、固证、风险研判、文书准备之间缺少一条连续的闭环

**我们给出的答案**

- 用浏览器插件把当前页面一键转成证据包
- 用工作台把案件、证据、监控、草稿串成统一入口
- 用后端 `HermesOrchestrator` 把抓取、研判、文书、通知编排成可扩展工作流

**一句话理解本项目**

> 先留证，再行动。先把侵权页面变成可管理、可审核、可继续推进的案件材料。
